Welcome to Para-Meter Project

Para-Meter project is a customizable portfolio optimization tool that utilizes machine learning models. Users can choose among various machine learning algorithms, integrate their data source to train our models, set their risk and return preferences, create their custom features, or choose from a predefined set of features to customize the optimization process. Para-Meter Project will give flexibility to the users, and it will have a user-friendly interface, which will spare them from writing code or dealing with technical details.
